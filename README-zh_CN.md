# Faker

一个基于 Mixin 的 Minecraft Forge 注入式模组。

[English](README.md) [简体中文]()

---

### 关于

这个模组基于 Minecraft Forge 1.8.9 和 Mixin 0.7.11-SNAPSHOT 。

### 用法

将模组拖进 mods 文件夹，享受吧~

前往 设置 -> 控制 来设定快捷键。

你可以按下 右Shift 来打开 点击界面 。

组按钮可以 右键 来折叠它们。

模块按钮可以 左键 来启用它，或 右键 来配置它。
